# 🦜 Parrot OS – Security, Privacy & Development

![Parrot OS](https://www.parrotsec.org/img/parrot-logo.svg)

**Parrot OS** is a free and open-source GNU/Linux distribution based on **Debian**, designed for **security experts**, **developers**, and **privacy‑conscious** users[reference:1].

## 🔐 Why Parrot OS?

- 🛡️ **Security‑First** – Comes with a full suite of penetration testing and digital forensics tools.
- 🔒 **Privacy Focused** – Includes built‑in anonymity tools and system hardening[reference:2].
- 💻 **Developer Ready** – Supports all major programming languages and development environments.
- 🧠 **Beginner Friendly** – Easier to start with than Kali, while still being powerful[reference:3].

## 📦 Editions

| Edition | Description |
|---------|-------------|
| **Security Edition** | Full hacking & pentest tools pre‑installed[reference:4] |
| **Home Edition** | Clean OS without security tools – install only what you need[reference:5] |
| **Raspberry Pi** | Complete Parrot experience on ARM‑based single‑board computers[reference:6] |
